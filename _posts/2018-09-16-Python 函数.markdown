# Python 函数

## 函数的定义与使用

```
def add(x, y):
    result = x + y
    return result
def user_print(code):
    print(code)
a = add(1, 2)
b = user_print('python')
print(a, b)
```


## 返回多个数值


```
def damage(skill_1, skill_2):
    damage1 = skill_1 * 3
    damage2 = skill_2 * 2 + 10
    return damage1, damage2
damages = damage(3, 6)
print(damages[0], damages[1])
print(type(damages))
```

```
def damage(skill_1, skill_2):
    damage1 = skill_1 * 3
    damage2 = skill_2 * 2 + 10
    return damage1, damage2
skill_1_damages, skill_2_damages = damage(3, 6)
print(skill_1_damages, skill_2_damages)
print(type(damage(3,6)))
```

## 函数的参数

```
# 必须参数，关键字参数
# 形式参数
# 实际参数 函数调用过程中的参数
# 关键字参数，与必须参数的区别在于函数的调用上
def add(y, x):
    result = x + y
    return result
c = add(y = 3, x = 2)
print(c)
```

